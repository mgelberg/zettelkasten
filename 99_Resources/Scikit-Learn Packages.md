[[Scikit-Learn]] packages to use for [[Machine Learning]]


| Module                    | Description                             | Common Use Cases                                             |
| ------------------------- | --------------------------------------- | ------------------------------------------------------------ |
| sklearn.linear_model      | Models that assume linear relationships | Linear regression, logistic regression, ridge/lasso          |
| sklearn.tree              | Decision tree-based models              | Classification and regression trees                          |
| sklearn.ensemble          | Ensemble methods combining models       | Random forest, gradient boosting, AdaBoost                   |
| sklearn.svm               | Support vector machines                 | Classification and regression with margin-based optimization |
| sklearn.neighbors         | K-nearest neighbors algorithms          | Classification, regression, anomaly detection                |
| sklearn.naive_bayes       | Naive Bayes classifiers                 | Text classification, spam detection                          |
| sklearn.cluster           | Clustering algorithms                   | KMeans, DBSCAN, hierarchical clustering                      |
| sklearn.decomposition     | Dimensionality reduction                | PCA, TruncatedSVD, NMF                                       |
| sklearn.preprocessing     | Data preprocessing tools                | Scaling, encoding, imputation                                |
| sklearn.model_selection   | Model evaluation and selection          | Cross-validation, grid search, train/test split              |
| sklearn.metrics           | Performance metrics                     | Accuracy, precision, recall, ROC AUC                         |
| sklearn.pipeline          | Pipeline creation and management        | Chaining preprocessing and modeling steps                    |
| sklearn.feature_selection | Feature selection tools                 | SelectKBest, RFE, variance threshold                         |

Supportive for [[Machine Learning (MOC)]] and model evaluation
